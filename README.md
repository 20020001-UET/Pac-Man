# SDL2 Template

Codeblock SDL2 project template.

## Version

Compiler **64-bit.**
SDL2 library: **x86_64-w64-mingw32**

## Setup

**Include header files:**

- SDL2
- SDL2_image
- SDL2_mixer

**Library files:** (/lib)

- SDL2
- SDL2_image
- SDL2_mixer

**Linker:**

- -lmingw32
- -lSDL2main
- -lSDL2
- -lSDL2_image
- -lSDL2_mixer
